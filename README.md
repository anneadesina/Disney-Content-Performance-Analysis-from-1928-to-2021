# Disney-Content-Performance-Analysis-from-1928-to-2021
This report analyzes how Disney+ content performed between 1928 and 2021, highlighting release trends, country production output, content type distribution, age ratings, top actors, top directors, and genre dominance.

## Introduction
The purpose of this analysis is to understand how Disney+ content evolved over time and identify the strongest performing categories on the platform.

The report focuses on:
- Top producing countries
- Most uploaded content types
- Popular age ratings
- Leading directors and actors
- Genre performance
- Content release trends over time

## Dataset Story
The dataset contains Disney+ titles from 1928 to 2021. Each row represents one title with details such as type, release year, country, rating, cast, director, and genre.

Main fields used:
- Title ID
- Title Name
- Content Type
- Country
- Release Year
- Rating
- Director
- Cast
- Genre

## Data Cleaning and Preparation
Before analysis, the data was cleaned by:
- Removing duplicates
- Standardizing country names
- Correcting content type labels
- Grouping similar genres
- Handling missing values
- Organizing years chronologically

## Key Findings
- Movies made up 72.55% of uploaded content
- TV Shows made up 27.45%
- USA was the top producing country
- 2021 was the most active release year
- Family was the top genre
- G, TV-G, and TV-PG were the leading ratings
- Content growth increased sharply after 2000

## Dashboard Visuals
The dashboard includes:
- Map chart for producing countries
- Donut charts for content type and ratings
- Bar charts for directors and actors
- Line chart for yearly content growth
- Treemap for top genres
- Interactive slicers for filtering results

## Recommendations
- Expand TV show production
- Continue investing in family content
- Increase international productions
- Use top actors and directors for future titles
- Study audience viewing behavior by genre

## Conclusion
Disney+ experienced strong content growth led by movies, family programming, and USA productions. The platform’s catalog reflects a strong family entertainment brand with room for more global expansion.
